# demoQR

Deze repo bevat nu twee aparte projecten:

- `demoqr-static`: de statische QR-variant
- `demoqr-dynamic`: de dynamische QR-variant

Beide projecten zijn zelfstandige Spring Boot apps met hun eigen Gradle wrapper.

## Runnen

Statische variant:

```powershell
cd demoqr-static
.\gradlew.bat bootRun
```

Dynamische variant:

```powershell
cd demoqr-dynamic
.\gradlew.bat bootRun
```
